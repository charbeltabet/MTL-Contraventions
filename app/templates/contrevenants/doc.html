/contrevenants

/contrevenants get

get

Returns an ordered list in descending order of establishments and their violations between the given dates. If dates are not provided, the list of all establishments with violations and their counts of violations is returned.

get /contrevenants

Returns an ordered list in descending order of establishments and their violations between the given dates. If dates are not provided, the list of all establishments with violations and their counts of violations is returned.

  • Request
  • Response

Query Parameters

  • du: (date)

    The start date to search from.

  • au: (date)

    The end date to search until.

/contrevenants/search get

get

Returns a list of violations matching the given search criteria. At least one search parameter must be provided.

get /contrevenants/search

Returns a list of violations matching the given search criteria. At least one search parameter must be provided.

  • Request
  • Response

Query Parameters

  • establishment: (string)

    The establishment name to search for.

  • owner: (string)

    The owner name to search for.

  • address: (string)

    The address to search for.
